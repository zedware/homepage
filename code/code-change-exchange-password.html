如何修改Exchange邮件帐号的密码

1、方法一

    将机器设置到Exchange服务器所在的域，然后修改域用户的登录密码。该密码也就是邮件帐户的密码。如果机器本来就在这个域内，操作起来还比较简单。否则还要修改域，重启机器，这样实在是太麻烦。

2、方法二

    在登录进入系统后，按下“Ctrl＋Alt＋Del”，在弹出的对话框中选择“更改密码”，在“更改密码”窗口中输入：

a）邮箱的用户名，例如：uniware。

b）要登录的域服务器IP地址，例如：192.168.1.1。

c）旧的邮件用户密码，新的邮件用户密码。

    点击“确定”后，耐心地等待一会儿即可。
