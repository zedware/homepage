MS Office 小技巧

    这些东西本来是不值得一写的，Google一下就能找到一大堆。

1. Outlook Express 无法打开邮件中的附件

    这个是因为 Outlook Express 中默认的安全设置在作怪。操作步骤如下：选择主菜单中的“工具”-> “选项”，在打开的“选项”对话框中选择“安全”页面。去掉“不允许保存或打开可能有病毒的附件”前面的对勾。

2. 更改 Outlook Express 中的邮件存储位置

    操作步骤如下：选择主菜单中的“工具”-> “选项”，在打开的“选项”对话框中选择“维护”页面。点击按钮“存储文件夹”，选择你所希望的存储路径。
