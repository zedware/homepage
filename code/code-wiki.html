哪个Wiki更适合你？

    Wiki的出现已经有几年了，目前已经得到了非常广泛的应用。在搜索某些词语时，Google给出的第一个结果可能就来自 Wikipedia。Wiki可以充分发挥人民群众的聪明才智，凝聚大家的力量。如果你想自己设置一个Wiki系统，也不是什么太难的事情。既有一些商业的Wiki产品，也存在着众多的开源软件。WikiMatrix提供了非常好的比较服务，可以全方位、多角度的进行各种系统的比较。有了它的帮助，选择一个合适的系统将变得相对简单一些。

    不过需要注意的是，多数开源的Wiki都是以Linux/Apache等为主要的支持平台，虽然在Windows/IIS上也可能可以运行，但是它们的设置都比较复杂，有些还需要MSYS或Cygwin等UNIX模拟环境。用得比较多的开源Wiki系统有：TWiki 、PhpWiki等。这里是两个支持Windows/IIS的Wiki系统：FlexWiki 和 Perspective ，你可以在线试试。

    Wiki系统一般都提供了Sandbox供大家测试和乱写乱画。不过不加什么限制的东西通常都会导致滥用。Wiki完全可能被人用它来写垃圾信息或者做广告，甚至被这些东西给撑死。要在那些特别出名的Wiki上颠倒黑白似乎比较困难，但要在那些影响力差一点的Wiki上混淆是非就比较容易了。不过这些东西早就不是什么新鲜事情了，我们已经在电子邮件、新闻组、BBS、Web论坛、留言板、Blog等地方见识过了，在短信、广播、电视、报纸、杂志、海报中，类似的事情似乎处处可见。生活就是这样多姿多彩，技术可以给生活带来便利，也会带来一定的烦恼。看来警察和监狱还是有必要存在的。
