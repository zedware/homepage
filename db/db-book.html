数据库教材

　　实际数据库系统采用的模型早已从最初的网状模型和层次模型过渡到今日流行的关系模型和对象关系模型，纯粹的对象模型还没有被广泛采纳。这里介绍几本学习数据库理论知识的入门教材。

　　数据库系统概论（第三版），萨师煊、王珊，高等教育出版社

　　数据库系统原理，王能斌，电子工业出版社

　　数据库系统原理，李建中、王珊，电子工业出版社

　　An Introduction to Database(Seventh Edition), C. J. Date，Addison Wesley/Pearson，机械工业出版社(中译本)

　　A First Course in Database Systems，Jeffrey D.Ullman，Jennifer Widom，Prentice Hall/Pearson，清华大学出版社(影印本)

　　Database System Concepts(Fourth Edition)，Abraham Silberschatz，Mcgraw-Hill，机械工业出版社(中译本、英文版)

　　Database System Implementation，Hector Garcia-Molina Jeffrey D. Ullman Jennifer Widom，Prentice Hall/Pearson，机械工业出版社(中译本) 

　　 SQL Performance Tuning, Peter Gulutzan, Trudy Pelzer, Addison Wesley
